# Пошаговая инструкция по установке APK-файла на Android-устройство

Данная инструкция поможет вам установить приложение в формате APK на ваше Android-устройство. Файл уже готов и находится на компьютере. Следуйте шагам внимательно.

> ⚠️ **Важно о безопасности**
> Установка приложений из неофициальных источников может нести риски. Убедитесь, что вы доверяете источнику APK-файла. Устанавливайте только файлы, полученные из надёжных мест.

## Шаг 1: Передача APK-файла на устройство

Выберите один из способов передачи файла `app-debug.apk` с компьютера на ваше Android-устройство:

### Вариант A: Через USB-кабель
1. Подключите устройство к компьютеру с помощью USB-кабеля.
2. На устройстве разрешите передачу файлов (может появиться уведомление — выберите "Передача файлов" или "MTP").
3. Скопируйте файл `app-debug.apk` с компьютера в папку "Загрузки" (Download) на устройстве.
4. Отключите устройство.

### Вариант B: Через облачные сервисы
1. Загрузите файл `app-debug.apk` в облачное хранилище (Google Drive, Dropbox, Яндекс.Диск и др.).
2. Откройте это же облако на устройстве и скачайте файл в папку "Загрузки".

### Вариант C: Через мессенджеры
1. Отправьте файл `app-debug.apk` через Telegram, WhatsApp или другой мессенджер на ваш телефон.
2. Сохраните файл в папку "Загрузки".

## Шаг 2: Включение установки с неизвестных источников

Перед установкой необходимо разрешить установку приложений из неофициальных источников:

1. Откройте **Настройки** на вашем устройстве.
2. Перейдите в раздел **Безопасность** или **Приложения** (название может отличаться в зависимости от производителя).
3. Найдите пункт **Неизвестные источники** или **Установка неизвестных приложений**.
4. Включите разрешение для приложения, через которое вы будете устанавливать APK (например, Файловый менеджер, Браузер, Telegram и т.д.).

> 🔐 **Рекомендация**: После установки приложения отключите эту опцию для повышения безопасности.

## Шаг 3: Установка приложения

1. Откройте **Файловый менеджер** на устройстве.
2. Перейдите в папку **Загрузки** (Download).
3. Найдите файл `app-debug.apk` и нажмите на него.
4. Подтвердите установку (может появиться предупреждение — нажмите "Установить").
5. Дождитесь завершения установки.
6. После установки нажмите "Готово" или "Открыть".

## Готово!

Приложение успешно установлено на ваше устройство. Вы можете найти его в меню приложений.

Если возникнут проблемы с установкой — убедитесь, что:
- Файл не повреждён
- На устройстве достаточно свободного места
- Разрешена установка с неизвестных источников

Успешной установки!
